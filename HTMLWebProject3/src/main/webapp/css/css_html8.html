이시언, '미역국+명품지갑' 기안84 깜짝 생일 파티…무한감동 '울컥' ('태계일주')[종합]

		18일 방송된 MBC '태어난 김에 세계일주(이하 '태계일주')에서는 가방 하나 달랑 메고 무작정 남미로 떠난 기안84와 
		그를 위해 지구 반대편으로 달려온 찐형 이시언, 여행 유튜버 빠니보틀의 현지 밀착 여행기가 그려졌다.
		dl날 기안84와 이시언은 아마존에서 피라냐 낚시를 하는 투어에 나섰다. 
		"월척을 한다면 로또 같은 꿈같은 이야기다"라며 기대감에 부푼 기안84는 "유일한 취미가 낚시다"라고 자신감을 드러내기도 했다. 하지만 생각과 달리 쉽게 입질이 오지 않았고, 
		그 사이 이시언이 먼저 피라냐 낚시에 입질을 느꼈다. 기안84는 피라냐를 기다리다 먼저 잡은 선장의 피라냐를 풀어줬다. 
		"네가 잡은 것도 아닌데 왜 풀어주냐"는 이시언의 타박에도 기안84는 
		"얼마나 꼬리를 잘 헤엄치면서 갔는지 아냐"라며 뭉클한 마음을 드러냈다.
